# LAST ZONE – iOS Native Swift Wrapper Project

This directory contains the full Swift Xcode project for the iOS app.

## Build Instructions (For macOS + Xcode)

1. Copy the entire folder `LAST-ZONE-iOS` to your Mac machine.
2. Open the file `LAST-ZONE-iOS.xcodeproj` in Xcode.
3. Select your development team in the project settings (Signing & Capabilities).
4. Build or export to a `.ipa` file.

## Cloud Building (For Windows Users)

To compile to a `.ipa` without a Mac:
1. Push this folder to a GitHub repository.
2. Sign up on **Codemagic.io** (free account).
3. Connect your repository and start the iOS build. It will output the `.ipa` to download.
